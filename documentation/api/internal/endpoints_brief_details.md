- **POST** `/file-browser/write` â†’ `write_file()` â€” Write file content. Body: `{ path, content }`.
- **POST** `/file-browser/mkdir` â†’ `make_dir()` â€” Create directory. Body: `{ path }`.

- **POST** `/file-browser/delete` â†’ `delete_item()` â€” Delete file or directory. Body: `{ path, recursive }`.
- **GET** `/file-browser/download` â†’ `download_file()` â€” Download file as attachment. Query param: `path`. Uses `send_file(..., as_attachment=True)`.
